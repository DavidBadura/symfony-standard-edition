Standard Edtion
===============

Features
--------

* symfony3 structur (as far as known)
* doctrine2 migrations
* npm assets management
* bootstrap3 framework

Installation
------------

```bash
composer install
npm install
```

Development
-----------

```bash
# shortcut to run gulp watch ;-)
npm start
```

Tests
-----

You should add your tests in the `tests` folder.

```bash
bin/phpunit
```
